
# useless-npm-module
a useless npm module

## Installation
Install via NPM:

```bash
npm install useless-npm-module

```

## Usage

#### javascript

```javascript

const myapp = require(".");

console.log(myapp.textgen("flipped", "ae")) //returns generated text

```

#### TypeScript

```typescript

import * as myApp from 'useless-npm-module';

console.log(myapp.textgen("flipped", "ae")) //returns generated text



```
